2. 如何处理响应式布局

了解几种不同的布局概念

  • 静态布局/固定布局(Static Layout) 示例

    特点：固定布局、固定宽度大小，屏幕宽度不够超出显示滚动条

  • 流式布局/液态布局(Liquid Layout) 示例

    特点：铺满屏幕、宽度随屏幕大小变化，布局不变

  • 自适应布局(Adaptive Layout) 示例

    特点：布局随屏幕大小变化，宽度不变

  • 响应式布局(Responsive Layout) 示例

    特点：液态布局和自适应布局的结合

哪些布局？静态，液体，自适应，或响应

实现手段：

  • css媒体查询
  • 宽度百分比
  • margin、padding百分比
  • em、rem、vw、vh
  • flex布局

3. 人机交互的处理

  • 字体图标是什么？

    像制作字体一样，把画好的图标用Unicode编码，像使用字体一样使用图标

  • 为什么要用字体图标？

    减少文件大小、减少请求链接数、可以像使用字体一样使用图标

  • 字体图标有什么不足？

    不支持多色

    解决办法？未来必热：SVG Sprite技术介绍

使用字体图标的三种方式

  • 直接使用Unicode编码
  • 使用写好样式的css
  • 使用svg

Firefox、Chrome、Safari 以及 Opera 支持 .ttf (True Type Fonts) 和 .otf (OpenType Fonts) 类型的字体。

Internet Explorer 9+ 支持新的 @font-face 规则，但是仅支持 .eot 类型的字体 (Embedded OpenType)。

2. 响应式的缺点，怎么样的项目适合响应式？

代码量大，功能复杂时实现难度大。博客类、小型门户网站比较适合
